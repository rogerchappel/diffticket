# DiffTicket temp change brief demo

## Demo promise

Show how DiffTicket converts an uncommitted local diff into an issue-ready
change brief with changed surfaces, churn, deterministic risk hints, and likely
verification commands.

## Recording flow

1. Run `bash demo/run-temp-change-brief.sh`.
2. Open the generated Markdown brief from the printed temporary directory.
3. Highlight the changed file list, churn summary, risk hints, and verification
   section.
4. Re-run the same demo with `--format json` output already generated by the
   script to show agent-readable handoff data.

## Grounded talking points

- `diffticket brief` reads local git diffs and package scripts.
- `--cwd` allows a caller to inspect another checkout without changing shells.
- `--format json` provides structured output for automations and agents.
- The tool does not call hosted services or transmit diff contents.
